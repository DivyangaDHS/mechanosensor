// PROJECT OVERVIEW

Bio-Inspired Tactile Sensing
for Robotic Navigation

Inspired by insect antennae and mammalian whiskers, this system achieves reliable contact detection even while the robot itself is in motion.

Echo State Networks

Reservoir computing framework used for real-time ego-motion cancellation from raw vibration sensor signals.

Bionic Antenna System

Multi-contact tactile antenna inspired by biological whisker systems for 3D object localization.

Real-Time Filtering

Online motion filtering pipeline separating ego-motion noise from genuine tactile contact signals.

3D Localization

Robust spatial mapping framework for unstructured environments including disaster zones and underwater habitats.

Research Papers

Key literature that forms the theoretical foundation of this project.

[1]
A Forward Model for an Active Tactile Sensor Using Echo State Networks

Dr. Nalin Harischandra, Volker Durr · 2012

Objective: Predict expected sensor reading of robotic antenna to cancel ego-motion Method: Echo State Network (ESN) — a recurrent neural network for chaotic time series Setup: Accelerometer on antenna tip + accelerometer on robot platform + servo position Key result: Successfully detected 34 true touch events from 35 disturbances with 100% accuracy Relevance: This is the core bio-inspired algorithm our project builds upon

View PDF
[2]
Prediction of Self-induced Mechanoreceptive Sensor Readings in An Insect-inspired Active Tactile Sensing System

Dr.Nalin Harischandra , Volker Durr· 2013

Objective: Improve contact distance estimation and material classification using the same antenna sensor Method: FFT-based dominant frequency analysis + SVM classifier (vs ANN benchmark) Key finding: Vibration mode changes at 80% of antenna length — causes non-linear distance readings near tip Hardware: MPU6050 accelerometer on acrylic tube + Raspberry Pi 3 + Pan-Tilt unit Key result: SVM achieved 99.2% accuracy vs 80.8% for ANN at lower computational cost Relevance: Direct predecessor project — same hardware platform and supervisors

Insect-Inspired Tactile Contour Sampling Using Vibration-Based Robotic Antennae

Thierry Hoinville1, Nalin Harischandra1, Andr´e F. Krause, and Volker D¨urr

Objective: Physically implement the Contour-net model on a real robot using vibration-based antennae Method: CPG-based control with Hopf oscillators; each contact triggers a 180° phase shift causing antenna to "bounce off" surfaces Platform: Two vibration-based antennae mounted on a robotic platform — same type as our project's setup Key result: Successfully traced 3D object contours and localized contact events in 3D space Bio-inspiration: Mimics stick insect (Carausius morosus) antennal searching behavior switching from broad search to local sampling on contact Relevance: Demonstrates the physical platform closest to our project — Pan-Tilt with vibration-based antenna and accelerometer

View PDF
[5]
Stable phase-shift despite quasi-rhythmic movements: a CPG-driven dynamic model of active tactile exploration in an insect

Nalin Harischandra, André F. Krause 2 and Volker Dürr · 2015

Objective: Model the Central Pattern Generator (CPG) driving rhythmic antennal movements of stick insects Method: Two-level CPG — Hopf oscillators for rhythm generation + Pattern Formation Network for joint kinematics (16 Fourier components) Key finding: SP joint must lead HS joint by exactly 10–30° to produce natural elliptical antenna tip trajectory — outside this range, the pattern collapses Proprioceptor ablation: Removing hair field sensors only changes joint amplitude and offset — the phase coupling remains stable (coded into CPG) Platform: 3D skeletal simulation of stick insect head-antennal system using Python + ODE physics engine Relevance: The CPG framework directly informs how our Pan-Tilt unit should move the antenna — bio-inspired oscillatory movement pattern for efficient contact detection

View PDF
[6]
Antennal Movements and Mechanoreception: Neurobiology of Active Tactile Sensors

Erich M. Staudachera, Michael Gebhardtby, Volker Du¨rr

Objective: Understand biological principles behind insect antenna-based tactile sensing and active exploration. Method: Studied antennal biomechanics, mechanoreceptors, active sensing behaviors, and sensorimotor control in insects. Key result: Showed that insect antennae can detect contacts, localize objects, and distinguish external stimuli during active movements. Bio-inspiration: Based on tactile exploration behavior of stick insects, locusts, cockroaches, and other antenna-based insects. Relevance: Provides the biological basis for vibration-based robotic antennae, contact detection, and ego-motion cancellation used in our project.
